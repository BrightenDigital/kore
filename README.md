## ğŸ”§ Working with Documentation (Developers)

This repository contains the official Compass AI documentation built with [VitePress](https://vitepress.dev/).

### Prerequisites

- Node.js 18+ and npm

### Local Development

```bash
# Clone the repository
git clone https://github.com/BrightenDigital/compass.git
cd compass

# Install dependencies
npm install

# Start the development server
npm run docs:dev
```

The documentation site will be available at `http://localhost:5173`

### Building for Production

```bash
# Build the documentation
npm run docs:build

# Preview the production build
npm run docs:preview
```
